# Portfolio Risk Return Analysis

This project performs portfolio optimization using Modern Portfolio Theory.

## Features
- Download historical stock prices using Yahoo Finance
- Compute daily returns
- Calculate portfolio risk and return
- Simulate 5000 portfolios
- Compute Sharpe Ratio
- Plot Efficient Frontier

## Technologies
Python  
NumPy  
Pandas  
Matplotlib  
yfinance

Best Portfolio Return: 0.0015602941832709037
Best Portfolio Risk: 0.021978936407548395
